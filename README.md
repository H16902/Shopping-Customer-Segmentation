This project analyzes mall customer data to understand spending behavior and group similar customers using K-Means clustering.

## Objective
To segment customers into meaningful groups based on **Age, Annual Income, and Spending Score** to support future marketing and business strategy.

## Approach
- Performed **Exploratory Data Analysis (EDA)** to understand customer distributions.
- Plotted **histograms and box plots** to find patterns, outliers, and key statistics like mean and median.
- Used **Minâ€“Max scaling** to normalize data before clustering.
- Determined the optimal number of clusters using the **Elbow Method**.
- Applied **K-Means clustering** to segment customers into **5 clusters**.
- Interpreted cluster behavior visually to identify different customer types.

## Key Insights
- Customer groups showed clear differences in spending habits and income levels.
- Segmentation can later help design targeted marketing strategies (to be added when project is expanded).

## Tools & Libraries
- Python

## Results
Customers were segmented into **5 clusters** based on normalized behavioral patterns.
